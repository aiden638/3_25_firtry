
## 평발 위험도 분석 웹사이트

참고 이미지(Fotor 스타일)를 기반으로, 깔끔하고 접근하기 쉬운 평발 분석 사이트를 구축합니다.

### 디자인 시스템
- **색상**: 민트/그린 계열 (#10B981, #059669) + 화이트 배경 + 다크 텍스트
- **스타일**: 깔끔하고 모던한 의료/건강 서비스 느낌
- **언어**: 한국어

### 페이지 구성

#### 1. 공통 상단 네비게이션 (고정)
- 로고 + 사이트명 ("FootCare AI" 또는 적절한 한국어명)
- 메뉴: 홈, 발 분석, 내 기록, 로그인/회원가입
- 스크롤 시에도 상단에 고정

#### 2. 홈페이지 (메인)
- **히어로 섹션**: 서비스 소개 타이틀 + 설명 + "분석 시작하기" CTA 버튼
- **서비스 소개**: 3단계 프로세스 안내 (사진 촬영 → AI 분석 → 결과 확인)
- **하단 푸터**: 간단한 정보

#### 3. 발 분석 페이지 (핵심)
- **중앙 대형 레이아웃** (참고 이미지처럼):
  - 왼쪽: 업로드된 사진 미리보기 영역 (큰 사이즈)
  - 오른쪽: 드래그 앤 드롭 업로드 영역 + 파일 선택 버튼
- **분석 결과 영역** (사진 업로드 후 표시):
  - 평발 위험도 점수 (게이지/프로그레스 바)
  - 위험 단계 표시 (정상 / 주의 / 위험)
  - 간단한 설명 텍스트
  - "전문가 상담 신청" 버튼 (추후 기능 연결용)

#### 4. 내 기록 페이지
- 로그인 필요 안내 (현재는 더미)
- 분석 이력 목록 (날짜, 점수, 상태)
- 개선 추이 차트 (목업)

#### 5. 로그인/회원가입 페이지
- 간단한 로그인 폼 UI (실제 인증은 추후 Supabase 연동)

### 주요 기능
- 이미지 드래그앤드롭 및 파일 선택 업로드
- 업로드 시 이미지 미리보기
- 더미 분석 결과 표시 (점수, 위험도 등)
- 반응형 디자인 (모바일 대응)
